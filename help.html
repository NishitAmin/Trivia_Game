How to play the game

This trivia game is created to test the basic concepts of java.
The player will face 10 multiple choice questions one after one
in which you have to select one of them and will get 1 point
for each correct answer. Select an answer and click "Next" button
to get the next question and click "Prev" button to see the previous
question. Click "Start Over" to reset the game.

Enjoy the Game !!!
